# Outil De Bruteforce Pédagogique

Un projet Python pour générer et cracker des mots de passe.

## Description

Ce projet pédagogique permet d'entrer ou générer aléatoirement des mots de passe plus ou moins sécurisés pour tenter de les 
cracker soit de manière native avec un code python soit de les hacher puis de les cracker avec Hashcat
en utilisant les méthodes de hachage suivantes : 
    - md5
    - sha1
    - sha256
    - sha512
    - bcrypt
